# Brand Profiles

Pre-configured settings for known client brands. When a user mentions one of these brands, load the corresponding profile automatically and confirm with the user.

If a brand is not listed here, ask the user for tone and any special preferences.

**Default separator for ALL brands: `|`**

---

## How to Read a Profile

Each profile contains:
- **Brand suffix**: Exact text to append in title (may differ from brand name)
- **Sector**: Industry vertical
- **Tone**: CTA aggressiveness level — Aggressive / Moderate / Soft / Informational
- **Language(s)**: Primary language(s) for content
- **Special notes**: Any brand-specific rules or exceptions

All brands use `|` as separator.
